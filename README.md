**App Features**
- splash screen
- onboard screen
- pin lock screen
- sign in sign up
- forgot password
- change password
- verification screen
- profile screen
- send money
- top up
- bill payment
- all transaction screen

**State Management**
- GetX

**Backend**
- Firebase is used for login and sign up. We also used Firebase storing users information.

**Security**
- pin encryption storing firebse
- Auto lock when there is no user activity
